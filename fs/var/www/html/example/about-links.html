Links are able to be traversed by using the "select links" command. To see all links available to select on a page, use the "show links" command, which will reveal the panel with a link of all links with the corresponding number to use to navigate to the link.

E.G. if the link panel shows:

  • 0 Home
  • 1 About
  • 2 Random Thoughts
entering "select link 1" will navigate to the "About" link.
